How to set up a webpage to be accessable through TOR?

this setup is for linux server devices!
1. install tor
- sudo apt install tor
2. in the configuration files for tor, make the following modifications
HiddenServiceDir /var/lib/tor/your_website_name
HiddenServicePort 80 127.0.0.1:80
// can specific the port based of http / https traffic
3. once the service is restarted, tor will automatically create
the hostname for your website on the dark web!

for here the setup differs depending on how you have your
webpage hosted, but here is how i did it with apache!
4. in apache, since I ONLY WANT it to display the website on the darkweb
make the following modifications to the configuration files.
/etc/apache2/sites-enabled/000-default.configuration
ServerName "insert onion address" "VirtualHost 127.0.0.1:80"
by doing the virtualhost 127.0.0.1, it makes it so that apache does not display
the webpage if followed by a domain name or ip address, but instead it defaults
to the tor setup, with onion address being the only addressable name to find the service!
